# ULoader
Universal media downloader for YouTube and other platforms, based on [yt-dlp](https://github.com/yt-dlp/yt-dlp) and [PyQt6](https://github.com/qt/qtbase).

## Note
> [!IMPORTANT]
> The executable file built from source code may trigger Windows Defender or other antivirus software. This can happen because the generated `.exe` file does not have a digital signature or reputation history. This is a false positive in many cases and does not necessarily indicate that the application contains malware.
> 
> To avoid this, you can build the project yourself from the source code or verify the build process and dependencies before running the executable.

## Current downloading features
- Video with different quality
- audio with thumbnail
- thumbnail
- json metadata
- GPU Video processing
- Download retries
